# 測試案例清單 — momo 首頁搜尋

對象：[momo 購物網](https://www.momoshop.com.tw/) 首頁搜尋區，分為**輸入框**、**搜尋按鈕**、**自動完成下拉**三部分。

範圍說明：本清單聚焦行為明確、可直接斷言的正向流程；邊界 / 規則未知的情境（空字串、特殊字元、長度上限、無匹配等）因站台規則尚不明確，暫不納入。

優先級：P0 = 核心必測，P1 = 重要，P2 = 加分。

## A. 輸入框 Input

| ID | 優先 | 測試案例 | 預期結果 |
|----|------|----------|----------|
| IN-01 | P0 | 輸入英文關鍵字 | 文字正確顯示在搜尋框內 |
| IN-02 | P0 | 輸入中文關鍵字 | 正確顯示、可送出 |
| IN-03 | P1 | 以 Enter 鍵送出（不點按鈕） | 等同按搜尋，導向搜尋結果頁 |
| IN-04 | P1 | 搜尋後關鍵字保留在框內 | 結果頁搜尋框仍顯示原關鍵字 |

## B. 搜尋按鈕 Button

|----|------|----------|----------|

## 搜尋結果頁 — 排序與篩選

> **重要前提（廣告排除）**：搜尋結果第一列為廣告商品（卡片帶「Ad」標記，且無評分 / 評價 / 總銷量資訊），會固定置頂、不隨排序或篩選改變。因此以下排序與價格篩選的斷言**僅針對非廣告的一般商品卡片**（第二列起），需先過濾掉廣告卡片再驗證。
>
> 價格取值：由卡片價格文字去除 `$` 與千分位逗號後轉為數值再比較。

### D. 排序 Sort

| ID | 優先 | 測試案例 | 預期結果 |
|----|------|----------|----------|
| SO-01 | P0 | 點「價格」排序 | 非廣告商品價格呈遞增（或首次點擊的預設方向）排列 |
| SO-02 | P1 | 再點一次「價格」切換方向 | 非廣告商品價格呈遞減排列 |
| SO-03 | P1 | 點「評價」排序 | 非廣告商品先依星等高→低排列；星等相同時再依評價數量多→少排列 |

### E. 價格區間篩選 Price Filter

| ID | 優先 | 測試案例 | 預期結果 |
|----|------|----------|----------|
| PF-01 | P0 | 輸入最低價＋最高價，按「確認」 | 所有非廣告商品價格落在 [最低, 最高] 區間內 |
| PF-02 | P1 | 只輸入最低價，按「確認」 | 所有非廣告商品價格 ≥ 最低價 |
| PF-03 | P1 | 只輸入最高價，按「確認」 | 所有非廣告商品價格 ≤ 最高價 |

共 16 條。
